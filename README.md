# Agro Pro

Sistema de gestão agrícola completo — v9.4

## Funcionalidades
- Dashboard com KPIs da safra
- Agro-Copilot (IA com GPT-4o)
- Alertas automáticos via WhatsApp
- Gestão de fazendas, talhões, safras
- Estoque de insumos e produtos
- Aplicações, colheitas, combustível
- Manutenção de máquinas
- Relatórios e exportação CSV
